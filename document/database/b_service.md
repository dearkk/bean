服务信息数据库表名：b_service,字段如下

---
| 字段标识 | 类型 | 必填  | 默认值  | 属性  | 字段名称
|:----------|:----------|:----------|:----------|:----------|:----------|
| id    | varchar(32)    | 是    | -    | primary_key    | 服务唯一标识ID   |
| application_id    | int   | 是    | -    | -    | 所属应用ID   |
| content    | varchar(4096)   | 是    | -    | -    | 创建信息  |
| created_at   | time | 是   | 自动生成   | -    | 创建时间    |
| update_at   | time | 是   | 自动生成   | -    | 上次更新时间    |
