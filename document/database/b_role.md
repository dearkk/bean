角色数据库表名：b_account,字段如下

---
| 字段标识 | 类型 | 必填  | 默认值  | 属性  | 字段名称
|:----------|:----------|:----------|:----------|:----------|:----------|
| id    | int    | 是    | 自增ID    | primary_key    | 唯一自增ID    |
| name    | varchar(16)    | 是    | -    | -    | 角色名称    |
| created_at    | time    | 是  | 自动生成  | -    | 创建时间   |
| desc    | varchar(128)    | 否  | -  | -    | 角色描述  |
