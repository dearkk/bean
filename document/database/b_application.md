应用信息数据库表名：b_application,字段如下

---
| 字段标识 | 类型 | 必填  | 默认值  | 属性  | 字段名称
|:----------|:----------|:----------|:----------|:----------|:----------|
| id    | varchar(32)    | 是    | -    | primary_key    | 应用唯一标识ID   |
| name    | varchar(64)    | 是    | -    | -    | 应用名称    |
| project_id    | int   | 是    | -    | -    | 所属项目ID   |
| created_at   | time | 是   | 自动生成   | -    | 创建时间    |
