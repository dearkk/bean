**简要描述：**

- 该接口前端展示菜单的角色。

**请求URL：**
- ` http://xx.com/grape/v1/menu/role/create `

**请求方式：**
- POST

**参数：**
```
  {
    "name":"应用管理", //菜单角色名称
    "desc":"",  //角色信息描述
  }
```


**返回结果**

``` 
  {
    "code": 200,    //200为正确返回，非200参考全局错误定义文档描述
    "msg": "success!"
    "data": {
      "result": {
        "name": "应用管理", //菜单角色名称
        "desc": "",  //角色信息描述  
        "create_at": "2020-01-01 12:30"  //创建时间  
      }
    }
  }
```

**备注**

- 更多返回错误代码请看首页的错误代码描述
